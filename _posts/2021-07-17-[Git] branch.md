~~*계속 덧붙이기*~~

### 브랜치 생성
```
$ git branch <new_branchname>
```
### 브랜치 전환
```
$ git checkout <branchname>
```
### 브랜치 병합
```
$ git merge <병합할branchname>
```

### 브랜치 삭제
```
$ git branch -d <branchname>
```
